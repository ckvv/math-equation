# math-equation

x、y方程可视化

## 二次方程

方程需要为`...x`的形式；如`x*x`,`4x`,`tan(x)`,`sqrt(1-x*x)`等：

例子: `cos(x);sin(x)`

![newplot](https://user-images.githubusercontent.com/30174970/127955775-152d0d90-53d3-4bfe-b71e-1f640fa66723.png)

## 三次方程

方程需要为`...xy`的形式；如`x*y`,`cos(x*y)`等：


例子: `cos(x);y*cos(y)`
